 layerJS 

lean, opensource UX composition library for modern websites and webapps

One concept - one library

General UX patterns like menus, sliders, layers & lightboxes, parallax effects, page-swipes, zoom effects are really just visual elements on interactive animated layers. layerJS provides one simple universal concept in one single javascript library to create such patterns in a most flexible way. 

How layerJS works - Stages & Frames

1.

Organize content in Frames

Instead of creating complete HTML pages with code for menues or sliders, simply design all parts of your site as separate static HTML & CSS fragements, the Frames.

2.

Stages define positions

Stages define the areas where the Frames (page-content, menu-panes, lightbox content, slides) will be displayed dynamically in your final page. 

3.

UI composition & state transitions

Frames are dynamically fit into Stages to compose the final user interface. Upon user interaction, the state of the user interface changes by exchange one or more Frames in the Stages.

Why layerJS is different

Just HTML

With layerJS the full layout and interactivity can be defined by pure CSS and HTML

A framework

layerJS does not just provide transition effects but provides a full framework for state and interactions, including gestures, links/url, and API

Full flexiblity

most libraries have one special effect and often limit design. layerJS just animates empty frames in empty layers so you can add every kind of design layout you want

Getting started

To use layerJS you just need to define the stages and put all content in frames. The following list gives you minimal examples for common UX patterns. Feel free to combine them to your liking.

Slider

A slider needs a single stage, a layer and one frame for each slide. Controls can be placed on a second overlapping layer. Controls are regular links that link to specific frames or use special links like "#!next" or "#!prev"

Sample Code

<div lj-type="stage">
  <div lj-type="layer" lj-default-frame="frame1">
    <div lj-type="frame" lj-name="frame1">
      ... content of frame 1 ...
    </div>
    <div lj-type="frame" lj-name="frame2">
      ... content of frame 2 ...
    </div>
    <div lj-type="frame" lj-name="frame3">
      ... content of frame 3 ...
    </div>
  </div>
</div>
Show documented code 
<html>
<head>
  <script src="https://cdn.layerjs.org/libs/layerjs/layerjs-0.4.0.js"></script> //Import layerJS javascript and css files.
  <link href="https://cdn.layerjs.org/libs/layerjs/layerjs-0.4.0.css" type="text/css" rel="stylesheet" />
  <link href="your-style.css" type="text/css" rel="stylesheet" /> //include your own styling for stage positions and visuals.
</head>
<body>
  <div lj-type="stage"> // Define a single stage.
    // Define the layer "slider" for the slides.
    // The default frame is "frame1"
    <div lj-type="layer" lj-name="slider" lj-default-frame="frame1" lj-native-scroll="false"> 
      // Define one frame per slide.
      // lj-fit-to="cover" will make sure that the stage will be fully filled with the frame.
      // "neighbors.l"/"r" define which frames will be transitioned to upon left/right swipe gestures.
      <div lj-type="frame" lj-name="frame1" lj-fit-to="cover" lj-no-scrolling="true" lj-start-position="center" lj-neighbors.r="frame2">
        <img src="img/samples1a.png">
      </div>
      <div lj-type="frame" lj-name="frame2" lj-fit-to="cover" lj-no-scrolling="true" lj-start-position="center" lj-neighbors.l="frame1" lj-neighbors.r="frame3">
        <img src="img/samples1b.png">
      </div>
      <div lj-type="frame" lj-name="frame3" lj-fit-to="cover" lj-no-scrolling="true" lj-start-position="center" lj-neighbors.l="frame2">
        <img src="img/samples1c.png">
      </div>
    </div>
    // A second layer for the controls
    <div lj-type="layer" lj-native-scroll="false" id="arrows">
      // The control frame. The frame will be centered in the stage.
      <div lj-type="frame" lj-fit-to="width" lj-start-position="center" lj-no-scrolling="true">
        // the link will trigger the transition to the previous frame of the currently active frame in layer "slider".
        <a class="btn-left btn" href="#slider.!prev?p=right">&lt;</a>
        <a class="btn-right btn" href="#slider.!right">&gt;</a>
      </div>
    </div>
  </div>
  <script>
    layerJS.init(); // initialize layerJS
  </script>
</body>
</html>

Parallax

Parallax effects can be created with two overlapping layers in a stage. If the frame in the upper layer is longer than the stage, it will scroll within the stage. Transparent areas in this frame will reveal the lower non-scrolling frame created the parallax effect.

Sample Code

<div lj-type="stage"> 
  <div lj-type="layer"> 
    <div lj-type="frame" lj-fit-to="cover"> 
      <img src="img/prezi.png">
    </div>
  </div>
  <div lj-type="layer">
    <div lj-type="frame"> 
    </div>
      ... content ...
  </div>
</div>
